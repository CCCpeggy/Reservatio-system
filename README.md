# Reservatio-system

軟體工程作業──會議室預約系統

## 使用專案

> 如果有不懂的部分就問我喔

### 下載專案

``` cmd
git clone https://github.com/CCCpeggy/Reservatio-system.git
```

### 寫專案前

* 先同步目前的內容

    ``` cmd
    git pull origin main
    ```

* 確定顯示 up-to-date 再開工

### 上傳目前進度

* 先同步目前的內容
    
    ``` cmd
    git pull origin main
    ```

* 確定顯示 up-to-date 再進行以下動作

    ``` cmd
    git add .
    git commit -m "進度描述"
    git push origin main
    ```
